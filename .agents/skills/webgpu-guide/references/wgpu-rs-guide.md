# wgpu (Rust) Guide

Based on wgpu.rs.

## Overview
`wgpu` is a cross-platform, safe, and pure-Rust graphics API. It implements the WebGPU standard, allowing code to run on Vulkan, Metal, DirectX 12, and OpenGL ES natively, as well as via WebAssembly in browsers.

## Key Features
- **Portability**: Runs on all major native platforms and the web.
- **Safety**: Provides safe abstractions over unsafe underlying graphics APIs.
- **Standards Compliant**: Closely follows the WebGPU specification.
